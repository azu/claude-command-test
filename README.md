# claude-command-test

Claude Codeの動作確認用テストリポジトリです。

## 概要

このリポジトリは、Claude Codeのコマンドやワークフローをテストするために使用されます。

**注意: ここはClaude Codeのサンドボックス環境です。** テストや実験のために自由に使用できます。

## ファイル構成

```
.
├── README.md      # このファイル
├── new-file       # テスト用ファイル
└── sample.txt     # サンプルテキストファイル
```

## 使い方

このリポジトリをクローンして、Claude Codeの各種機能をテストできます。

```bash
git clone https://github.com/azu/claude-command-test.git
cd claude-command-test
```
